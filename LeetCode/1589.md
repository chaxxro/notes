# 区间次数统计

区间统计

```cpp
/*
使用差分数组统计区间次数，利用差分数组统计频率变化导数

diff[i] 表示第 i 个元素的查询频次与第 i - 1 个元素查询频次差值

对于 [start, end]，diff[start] += 1，diff[end + 1] -= 1;

对于元素频率 f[i] = f[i - 1] + diff[i]
*/
```
